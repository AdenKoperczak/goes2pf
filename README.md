# goes2pf
A simple script which downloads GOES imagery and saves it as a placefile for Supercell WX

## Installing Linux
1. Clone this repo.
1. Install `requirements.txt`
```bash
pip3 install -r requirements.txt
```
1. Run `goes2pf.py`
```bash
python3 goes2pf.py
```

## Installing Windows.
1. Download the latest version from the realeases.
1. Extract the files to an install folder.
1. Run `goes2pf.exe`

## Adding To Supercell WX
